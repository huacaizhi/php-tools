### 收集整理一些常用的函数类库

#### `DateTime`
> `msecTime`返回当前的毫秒时间戳


#### `Guid`
> `guid`创建一个 Guid


#### `Http`
> `getRequestUrl`返回一个完整的请求地址 URL

#### `Validation`
> Cli
>> `isCli`判断是否是在`cli`(命令行)模式下运行

>> `isCgi`判断是否是在`cgi`模式下运行

> Form
>> `isEmail`判断是否是`email`地址

>> `isUrl`判断是否是`url`地址

>> `isVersionNumber`判断是否是一个版本号
